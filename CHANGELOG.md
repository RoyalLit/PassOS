# Changelog

All notable changes to this project will be documented in this file.

## [0.2.0] - 2026-05-12
### Added
- Professional GitHub documentation suite (Templates, Contributing, Roadmap).
- Advanced Mermaid diagrams for architecture and pass lifecycles.
- Consolidated Security and API documentation.
- n8n workflow documentation for automated escalations.

## [0.1.5] - 2026-04-21
### Added
- SMS-based Parent Approval integration (Twilio).
- Forensic Audit Logging engine for administrative actions.
- Multi-hostel support for Warden profiles.

## [0.1.0] - 2026-04-10
### Added
- Initial Multi-Tenant SaaS architecture.
- JWT-signed QR pass generation and scanning.
- Core Portals: Student, Guard, Admin, and Parent.
- Row Level Security (RLS) enforcement across all tables.

## [0.0.1] - 2026-03-15
### Added
- Project initialization with Next.js 16 and Supabase.
- Basic pass request schema.
